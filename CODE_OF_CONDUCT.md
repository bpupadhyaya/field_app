# Code of Conduct

## Our Pledge

We are committed to making participation in this project a harassment-free experience for everyone.

## Our Standards

Examples of behavior that contributes to a positive environment:

- Being respectful and constructive in feedback
- Assuming positive intent and asking clarifying questions
- Focusing on technical merit over personal preference

Examples of unacceptable behavior:

- Harassment, insulting language, or personal attacks
- Trolling, deliberate disruption, or bad-faith arguments
- Publishing private information without explicit permission

## Enforcement

Project maintainers are responsible for clarifying and enforcing this Code of Conduct. They may remove, edit, or reject comments, commits, issues, and other contributions that violate these standards.

## Reporting

To report unacceptable behavior, contact the maintainers by opening a private security advisory or by emailing the repository owner listed in `SECURITY.md`.

## Scope

This Code of Conduct applies within all project spaces, including issues, pull requests, discussions, and external communication representing the project.
